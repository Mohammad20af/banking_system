# Mini Banking System 

A simple banking system project written in **Python**, using **OOP (Object-Oriented Programming)** principles and JSON for data storage.  
Perfect for a portfolio or as a practical learning project.

---

# Features 

- User registration (Create Account)
- User login (Login)
- Deposit and Withdraw with balance checks
- View Transaction History per user
- Data persistence using JSON
- Display top balances across all users
- Organized using **Modules & Packages**:
  - `models/` → Classes (`User`, `Account`)
  - `utils/` → Helper functions (load/save data, user management)
  - `data/` → JSON file storage
